# User Dashboard
![](/Screenshot.png)

## Setup
To run this project, install it locally using npm:

```bash
$ cd user-dashboard-vue
$ npm install
$ npm run dev
```

## Technologies
Project is created with:
* Vue JS
* Tailwind
* Vuetify
